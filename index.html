How to Apply CSS to HTML

To apply an external CSS file in an HTML file, use the link element inside the head section of the HTML page and include a reference to the external style sheet.

For example:

<head>
<link rel="stylesheet" href="mystyles.css"
</head>

How to Use JavaScript in a Webpage

To link an external JavaScript file, use the script element with a source attribute pointing to the script file.

For example:

<script src="path/to/my/script.js"></script>

To write JavaScript within HTML, you can write script code betweeen script tags instead of using a source attribute.

For example:

<script>
    window.addEventListener("load", () => {
        console.log("This function is executed once the page is fully loaded");
    });
</script>
